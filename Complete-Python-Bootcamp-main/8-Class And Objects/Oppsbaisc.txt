Pattern 1: LLM Model
class LLM:
    def __init__(self, model_name):
        self.model_name = model_name

    def generate(self, prompt):
        print(f"{self.model_name} generated: {prompt}")
Object Creation
llm = LLM("gpt-4o")
llm.generate("Explain AI")

Behind the scenes

model_name (parameter)
        │
        ▼
"gpt-4o"
        │
        ▼
self.model_name

Pattern 5: Prompt Template
class PromptTemplate:
    def __init__(self, template):
        self.template = template

    def format(self, question):
        print(self.template.replace("{question}", question))

Usage

prompt = PromptTemplate("Answer this: {question}")
prompt.format("What is AI?")

Pattern 14: Multi-Agent System

    def __init__(self, role):
        self.role = role

    def work(self):
        print(f"{self.role} is working")

Usage

coder = Agent("Coder")
reviewer = Agent("Reviewer")

planner.work()
coder.work()
reviewer.work()
